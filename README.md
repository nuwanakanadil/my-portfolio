## AI Portfolio Assistant

To enable the real AI response mode, create a `.env.local` file in the project root and add:

```bash
OPENAI_API_KEY=your_openai_api_key_here
OPENAI_MODEL=gpt-4o-mini
```

If the key is not set, the assistant falls back to the built-in local response engine.
`npm run dev` now starts both the frontend and the backend route at `/api/assistant`.
